# ASF-Tools
小工具

### 项目安装

```
npm install
```

### 启动
```
npm run serve  热启动 -默认浏览器打开
```

### 项目打包
```
npm run build
```


### 提交规范

Header Header 部分只有 1 行，格式为<type>: <subject>。

type 用于说明提交的类型 subject 用于概括提交内容。

1. feat：新功能（feature） 

2. fix：问题修复 

3. style：调整格式（不影响代码运行）
